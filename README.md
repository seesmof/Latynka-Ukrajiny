### package usage
#### in short
just use `convert(your_text)`, this will return your text in latin

#### in detail
the main function of the library is pretty much `convert`. it takes in a Ukrainian text in cyrillic alphabet and converts it into latin (romanizes, makes into latynka)

##### `Romanizer` class
you can also use the underlying class called `Romanizer` that the `convert` function uses. it contains a method called `romanize` that performs the latynkification process and returns the result back to you ‚úùÔ∏èüôèüèº

```python
r = Romanizer("–Ñ–î–ò–ù–ò–ô –ë–û–ì –Ü–°–£–° –•–†–ò–°–¢–û–° –°–≤—è—Ç–∏–π –ì–û–°–ü–û–î–¨ —ñ –°–ø–∞—Å–∏—Ç–µ–ª—å")
r.romanize()
```
```
JEDYNYJ BOG ISUS HRYSTOS Svjatyj GOSPOD' i Spasytel'
```

the `Romanizer` class's constructor takes in the same Ukrainian cyrillic text string and the `romanize` method converts it into latin

### latynka info
the romanization system was taken from Maks Prudeus: [the original video](https://youtu.be/nHeE2x2UNw4) and [a useful article](https://drukarnia.com.ua/articles/ukrayinska-latinka-maibutnye-nashoyi-krayini-ITKqx)

its pretty much the system from [Naƒåasi](https://nachasi.com/manifest/), but
- use `JI` in place of `–á` 
- use apostrophe `'` sign in place of `–¨` and `'`
- a different `ƒú` in place of `“ê`

full letters concordance can be found at `./latynka/concordance.json`

the initial `.JSON` concordance schema was taken [from here](https://github.com/paiv/latynka/blob/main/src/data/bundled_tables/nachasi.json) ‚úùÔ∏èüíì GOD bless you brother, i pray and hope you dont mind
